## ??? **TROUBLESHOOTING**

### **If App Doesn't Launch (F5)**
1. Right-click project ? **"Set as Startup Project"**
2. **Clean Solution** (Build menu)
3. **Rebuild Solution**
4. Press F5 again

### **If Code Doesn't Compile in MT5**
1. Check MT5 build number (needs 3661+)
2. Ensure you pasted in **new EA file** (not indicator)
3. Check for syntax errors in **Toolbox** tab
4. Save file before compiling

### **If EA Doesn't Trade**
1. Check **"Auto Trading"** button is enabled (green)
2. Verify **"Allow Algo Trading"** in EA properties
3. Check **"Expert Advisors"** tab for errors
4. Ensure strategy conditions are being met
